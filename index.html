Remet dans la poche organisateur les mises encaissées qui n'ont pas été créditées (régularisation). C'est le stock qui sert à payer les cagnottes — différent de la poche joueuse.
🚧 Mode maintenance

📩 Message privé (par code)
Visible UNIQUEMENT par le code indiqué (organisateur ou joueur), à sa connexion. Vider et publier pour retirer.
🚫 Bloquer un code joueur
Code compromis ou litige : la connexion et les pions deviennent inaccessibles. Réversible.
✉️ Envoyer un code joueur par email
Email de bienvenue avec le code, le cadeau de 2 000 XPF et les liens
📢 Message aux joueurs

📢 Message aux organisateurs
S'affiche en bandeau chez TOUS les organisateurs à leur connexion. Vider et publier pour le retirer.
🚀 Recréer les tickets en masse
Après un reset : une ligne = un ancien code (réactivé tel quel) ou +Nom (nouveau code généré). Les pions restent attachés aux codes.
🎁 Crédit en masse (dédommagement)
Crédite plusieurs codes d'un coup. Un code par ligne.
💳 Paiements Stripe — Rapprochement
Tous les paiements de pions reçus par carte, directement depuis Stripe. Cliquez « Créditer » après vérification — impossible de créditer deux fois.
🧰 Données & sauvegarde

🔍 Vérifier les codes émis par un organisateur
En cas de conflit : voir tous les codes joueurs qu'un organisateur a vraiment créés, avec les dates.
🔗 Fusionner deux codes organisateur
Transfère TOUT (pions, joueuses, ventes...) du code source vers le code cible. ⚠️ Fais une sauvegarde avant !
